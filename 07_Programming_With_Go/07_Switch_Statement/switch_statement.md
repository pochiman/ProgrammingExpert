# Key Term

# [Switch_Statement]
In programming, a `switch statement` is a type of selection control mechanism used 
to select a specific case to run based on the value of a variable or expression.

`var a int = 10`
`switch a {`
    `case 10:`
        `fmt.Println("a is 10")`
    `default:`
        `fmt.Println("a is not 10")`
    `}`
